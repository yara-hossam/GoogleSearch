This Project is for testing Google search by using Selenium with python.

The implemented test cases are:

1-For Google's homepage :
  a-Opening the correct homepage. (google's homepage itself)
  b-Search textbox exists.
  c-Search textbox maximum length = 2048.
  d-Gmail's icon correctly navigates to gmail's url.
  e-Images' icon correctly navigates to google images page.

2-For searching:
  a-Clicking the search button without inserting text doesn't redirect to another page.
  b-The page we reached satisfies the search text requirement. 
  c-The list of suggesstions appearing when typing in the search box has matching words with the typed text.
